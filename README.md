# 旅行手账

轻量级离线旅行手账，记录游记、管理开支。纯前端 PWA，数据仅存储在本地浏览器。

## 功能

- 📓 **游记** — 按时间线记录旅行见闻，支持滑动删除
- 💰 **记账** — 多币种支出管理，分类、支付方式、经手人、受益人，本位币换算
- 👥 **同行人** — 添加旅行伙伴
- 📊 **统计** — 支出分布饼图、人均支出、结算清单
- 🧾 **结算清单** — 自动计算每人实付/应担/净额，生成转账方案
- 🔗 **一键分享** — 生成链接，发给好友直接导入数据
- 🎨 **主题颜色** — 6 种主题色
- 💾 **导入/导出** — JSON 备份与恢复

## 更新记录

### v1.0.3

1. **本位币换算系统**
   - 在设置页选择本位币（如 CNY）
   - 记账时可选多种外币（USD、JPY、EUR、THB、VND 等）
   - 未设置汇率的支出，在统计页以红色标记
   - 在统计页批量设置汇率，自动换算为本位币

2. **统计页汇率批量设置**
   - 显示每种外币的未换算笔数
   - 填写汇率后一键应用到所有该币种支出
   - 解决"记账时不知道汇率"的实际问题

3. **支出列表支持修改**
   - 点击支出条目即可编辑
   - 支持修改金额、币种、类别、支付方式、经手人、受益人、日期、备注

### v1.0.2

1. **支出表单新增「受益人」选择**
   - 记一笔支出时，除了选"谁付的"（经手人），现在还可以选"这笔钱给谁花的"（受益人）
   - 默认全选所有人，可以手动取消某个人的勾选
   - 输入金额后会实时显示"3人均摊，人均 ¥xxx"

2. **统计页新增「结算清单」**
   - 只在"全部"人员模式下显示
   - 列出每个人的：实付金额、应担金额、净额
   - 正数（绿色）= 别人该转给你，负数（红色）= 你该转给别人
   - 自动生成转账方案

3. **支出列表显示分摊信息**
   - 如果不是全员均摊的支出，会在列表里显示"分摊：2人（张三、李四）"

4. **向后兼容**
   - 之前记的支出没有受益人字段，会自动当作"全员均摊"来处理
